&par Example Description 
The 'Blinky' project is a simple CMSIS RTOS based example for
Geehy Microelectronics 'APM32E103xx' microcontroller using Geehy APM32E103xx MINI Board,
compliant to Cortex Microcontroller Software Interface Standard (CMSIS v2.0).

Example functionality:
 - Clock Settings:
   - XTAL    =            8.00 MHz
   - SYSCLK  =           72.00 MHz
   - HCLK    = SYSCLK  = 72.00 MHz

 - LEDs are blinking as running light

&par Directory contents 
  - APM32E103MINI/Blinky/Source/main.c                      Main program

&par Hardware and Software environment
  - This example runs on APM32E103_MINI Devices.
